# Web3Proof 答辩演示脚本

## 前置条件

1. 配置 `deploy/.env`：`BACKEND_WALLET_PRIVATE_KEY`、`ADMIN_WALLET_ADDRESS`、`PINATA_JWT`（可选）
2. 运行 `scripts\deploy-sepolia.bat` 部署合约到 Sepolia，或使用已部署合约地址
3. MetaMask 切换 Sepolia，账户有测试 ETH
4. 启动后端与前端

## 演示流程

### 学生用户（账户 A）

1. 连接钱包登录
2. **个人资料** -> 填写姓名/昵称、成果主页、简介 -> **注册链上 DID**
3. **成果存证** -> 新建成果 -> 填写课程项目、竞赛证书、实习证明等材料 -> 上传 IPFS
4. 在成果详情中点击 **链上存证**，通过 MetaMask 完成链上登记
5. **成果认证** -> 选择成果类别 -> 提交材料说明与证明链接

### 管理员（ADMIN_WALLET_ADDRESS 账户）

1. 登录后进入 **管理后台 -> 成果认证审核**
2. 查看学生提交的成果认证申请
3. 审核通过申请
4. 点击 **发放 SBT**，通过 MetaMask 铸造不可转让认证凭证

### 核验方（账户 B）

1. **核验方注册** -> 填写单位/机构信息
2. **成果核验** -> 输入学生钱包地址 -> 查看核验报告
3. 核验报告展示 DID、成果数量、认证凭证数量、可信评分、综合评级和链上校验信息

### 公开展示

- 访问 `/portfolio/{学生钱包地址}` 查看可信成果档案
- 旧路径 `/resume/{学生钱包地址}` 仍兼容可访问
